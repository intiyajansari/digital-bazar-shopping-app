# 🛒 Digital Bazar - Shopping App

Digital Bazar ek online shopping application hai jisme users products ko buy aur sell kar sakte hain easily.  
Ye project basic e-commerce functionality ko demonstrate karta hai.

---

## 📌 Features

- 🛍️ Product Listing
- 🔍 Product Search
- 🛒 Add to Cart
- 💳 Checkout System
- 👤 User Registration & Login
- 📦 Order Management
